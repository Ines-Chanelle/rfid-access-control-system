# rfid-access-control-client-server-udp
Un système de contrôle d'accès RFID basé sur une architecture client-serveur UDP. Le client embarqué utilise Arduino Uno et ESP8266 pour la lecture RFID et le contrôle physique, tandis que le serveur Go assure le traitement des accès. la supervision temps réel et la gestion via SQLite.
